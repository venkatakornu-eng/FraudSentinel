5-Agent Workflow
Agent 1 — Stream Simulator
Generates batched transaction feeds at configurable intervals, mimicking Kafka/API ingestion.
Agent 2 — Risk Classifier
Scores each transaction via Random Forest (AUC ~0.91+). Returns probability, score 0–100, tier.
Agent 3 — Feature Explainer
Extracts top risk signals per transaction. LIME-proxy approach — transparent, no black box.
Agent 4 — Claude LLM Narrator
Calls Anthropic Claude API to generate professional analyst-ready alert narratives. Rule-based fallback.
Agent 5 — Alert Dispatcher
Packages JSON alerts, logs to file, routes to analyst queue or block system.
